# متطلبات بوابة التدريسين - تحليل متكامل

## 📋 الوظائف الأساسية (المطلوبة)

### 1. إدارة الطلاب
- عرض قائمة الطلاب حسب المادة
- تصفية حسب المرحلة (1، 2، 3، 4)
- تصفية حسب القسم
- عرض تفاصيل الطالب
- دعم التدريسي الذي يدرس في أقسام متعددة

### 2. إدارة الدرجات
- **امتحان الشهر الأول** (نظري + عملي)
- **امتحان الشهر الثاني** (نظري + عملي)
- **امتحان الشهر الثالث** (اختياري)
- **السعي الفصلي** (للفصل الأول/الثاني)
- حساب التقديرات تلقائياً
- إمكانية التعديل والتصحيح

### 3. الحضور والغياب
- تسجيل يومي لكل محاضرة
- حالات: حاضر ✅ | غائب ❌ | مجاز 📝
- التعرف على الطالب بالصوت (Voice Recognition)
- تقارير الحضور والغياب
- إحصائيات (نسبة الحضور لكل طالب)

---

## 💡 الاقتراحات الإضافية (للتطوير الاحترافي)

### 4. إدارة المواد التدريسية
- عرض جميع المواد التي يدرسها التدريسي
- تفاصيل كل مادة (المرحلة، القسم، عدد الطلاب)
- إحصائيات سريعة لكل مادة

### 5. التقويم الأكاديمي
- عرض الجدول الأسبوعي للمحاضرات
- تواريخ الامتحانات المهمة
- التذكيرات والإشعارات

### 6. التواصل مع الطلاب
- إرسال إشعارات للطلاب
- رسائل جماعية أو فردية
- نشر الإعلانات والتنبيهات

### 7. التقارير والإحصائيات
- تقرير الحضور والغياب الشهري
- تقرير الدرجات (المعدل، النجاح، الرسوب)
- إحصائيات الأداء حسب المادة
- تصدير التقارير إلى Excel/PDF

### 8. الملفات والمواد التعليمية
- رفع المحاضرات والملفات
- مشاركة روابط الموارد التعليمية
- تنظيم المواد حسب المادة والفصل

### 9. إدارة الامتحانات
- تحديد مواعيد الامتحانات
- إنشاء جداول الامتحانات
- إشعارات للطلاب قبل الامتحان

### 10. التقييمات والاستبانات
- استطلاعات رأي الطلاب
- تقييم أداء المادة
- ملاحظات الطلاب

## 🗄️ البنية التحتية المطلوبة

### جداول قاعدة البيانات الجديدة

1. **teacher_subject_assignments**
   - ربط التدريسي بالمواد التدريسية
   - دعم الأقسام المتعددة

2. **attendance_records**
   - سجل الحضور والغياب اليومي
   - ربط بالمحاضرة والطالب

3. **monthly_exams**
   - امتحانات الأشهر (1، 2، 3)
   - الدرجات النهائية

4. **lecture_schedule**
   - جدول المحاضرات
   - الأيام والأوقات

## 📂 هيكل الصفحات المقترح

```
/teachers-portal/
├── dashboard              (لوحة التحكم الرئيسية)
├── my-students           (طلابي - قائمة بالمواد والطلاب)
│   ├── [subject-id]      (تفاصيل مادة مع قائمة الطلاب)

│   ├── daily             (تسجيل يومي)
│   └── reports           (التقارير)
├── subjects              (موادي التدريسية)
├── calendar              (التقويم الأكاديمي)
├── communications        (التواصل مع الطلاب)
├── reports               (التقارير والإحصائيات)
├── exams                 (إدارة الامتحانات)
└── files                 (الملفات والمواد)
```
